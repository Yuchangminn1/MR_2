## Install
---

```
mkdir -p ~/catkin_ws/src
cd ~/catkin_ws/src
git clone https://github.com/ggh-png/MR_2.git
```
## Essential pakage install
---
```
cd ~/catkin_ws/src/MR_2
./essential_package_install.sh
source ~/.bashrc
```

## rebuild all
---
```
cd ~/catkin_ws/src/MR_2
./rebuild_all.sh
```
## build all
---
```
cd ~/catkin_ws/src/MR_2
./build_all.sh
```
## launch ui full size
---
```
cd ~/catkin_ws/src/MR_2
./launch_ui.sh
```
## launch ui max size
---
```
cd ~/catkin_ws/src/MR_2
./launch_ui_size_max.sh
```
